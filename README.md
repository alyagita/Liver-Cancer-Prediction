# Liver Cancer Prediction Using Machine Learning

## Overview
This project analyzes the impact of COVID-19 on liver cancer patients using clinical dataset data and builds machine learning models to predict cancer diagnosis based on patient characteristics.

---

## Key Contributions
- Cleaned and preprocessed dataset (missing values and outliers handling)  
- Performed exploratory data analysis (EDA)  
- Applied feature selection using PCA and Chi-Square  
- Built classification models (Naive Bayes, SVM, KNN, Neural Network)  
- Evaluated model performance using accuracy, precision, recall, and F1-score  
- Compared results with and without feature selection  

---

## Outcome
- All models achieved high classification performance  
- Neural Network and Naive Bayes showed consistently strong results  
- Feature selection improved model interpretability  
- Demonstrated that clinical data can effectively support liver cancer prediction
